The Device Setup Dialog

Description
Enter a descriptive title for your device.

Address
Enter the device IP address. A device name is also allowable if you have set it up with your router.

Package name, default: org.xbmc.kodi.
Select the package name by choosing your media center software from the Media Center presets, or enter package name directly.

If you are running Kodi leave this setting unchanged.

File Path, default: files/.kodi
Change the file path by choosing a selection from the Media Center presets or enter your own.

The period "." that you see at the beginning of "files" is very important. Don't leave it off!

The File Path is not set up on your Fire Device until Kodi has completed its initial run. adbLink uses the File Path in conjunction with the Package Name to locate the Media Center on your device. If either are entered incorrectly, none of the "Kodi" functions will work.

Media Center presets
4 presets are provided for Package name and filepath:
Kodi
SPMC
TVMC
Other (your package)
Select a preset with the Media Center radio buttons. The default is Kodi.

Pull Folder, default: User's home directory.
Choose the folder where pulled Kodi files and screenshot pngs are placed.

AFTV updates Toggle AFTV updates on/off. This disables/enables com.amazon.dcp. Default is updates on. Changing this option requires root.

SSH server at boot When checked this option has the SSH server start at boot. This option requires root.

The Check for updates at startup checkbox causes adbLink to check jocala.com for updates at startup. This is on by default.

The Check for updates button contacts jocala.com for an immediate update check. If an update is available adbLink offers to connect for download.
